<!-- DESCRIPTION/ -->

Adds support for the LESS CSS pre-processor to DocPad

<!-- /DESCRIPTION -->


<!-- INSTALL/ -->

<h2>Install</h2>

Install this DocPad plugin by entering <code>docpad install less</code> into your terminal.
